Bantuan Cetak (RawBT)

Aplikasi ini menggunakan layanan cetak RawBT untuk mencetak struk via Bluetooth. Pastikan aplikasi RawBT sudah terinstall di perangkat Anda.

Install RawBT dari Play Store
  1. Install Aplikasi RawBT.

    Unduh dan install aplikasi "RawBT print service" dari Google Play Store.

  2. Nyalakan Printer & Pasangkan (Pairing).

    Nyalakan printer thermal Anda. Buka Pengaturan Bluetooth di perangkat Android Anda, cari nama printer (misal: RPP02N, MPT-II), dan pasangkan. Jika diminta PIN, biasanya adalah "0000" atau "1234".

  3. Konfigurasi Printer di RawBT.

    Buka aplikasi RawBT, masuk ke pengaturan, dan pilih printer Bluetooth yang sudah Anda pasangkan tadi.

  4. Mulai Mencetak.

    Saat Anda menekan tombol cetak di aplikasi POS ini, sistem akan meminta Anda untuk membuka tautan dengan RawBT. Pilih RawBT dan struk akan tercetak.

Troubleshooting:

  • Pastikan printer sudah terpasang dengan benar di pengaturan Bluetooth Android.
  • Pastikan printer yang benar sudah dipilih di dalam aplikasi RawBT.
  • Coba matikan dan nyalakan kembali printer dan Bluetooth HP Anda.
  • Fitur ini hanya berfungsi di Android dengan RawBT terinstall.
